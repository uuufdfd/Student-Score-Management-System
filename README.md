# Student Score Management System

A Student Score Management System based on Qt and C++.  
The system is designed to manage student information and academic scores efficiently. It supports adding, editing, deleting, searching, sorting, and saving student records.

## Features

- Add student information
- Edit student records
- Delete student records
- Search students by ID, name, or class
- Sort student scores
- View score statistics
- Save student data to JSON file
- Simple and user-friendly Qt interface

## Technology Stack

- C++
- Qt Widgets
- JSON data storage
- Qt Creator
